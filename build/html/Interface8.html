Load Status

The load status button gives a visual indication of GARP's runtime status. While the system is waiting for input this button label displays the version of GARP you are running. After the user clicks "Display" from a dialog or makes some other data request this label changes to indicate GARP is processing the user's input request.

Click this load status button to pop up the GARP logging window. You can also access this logging window and adjust the logging window preferences by clicking Options on the main menubar and then Logging.
