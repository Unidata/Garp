GARP Main Menubar

Main Menubar

The main menubar gives you access to all the features in GARP. Other buttons under this menubar give you quicker access to a subset of the features available in the application. Starting from the left side of the menubar, we'll step through each dropdown menu and discuss the capabilities each button provides.

  • "Save" allows you to save the current graphics as GIF files. You can save either a single frame in a loop or the entire loop.
  • "Quit" allows you to quit GARP.

"View" contains menu items for selecting particular data types. This provides an alternate means of performing actions if the data type icons have been turned off.

From the buttons available under the "Area" dropdown menu, you can choose a predefined geographic region which will be drawn in plan view displays.

With the "Time" dropdown menu you can adjust time-related parameters.

  • "Auto-Update" allows you to toggle auto-update on/off and set the number of minutes between auto-update cycles. See the Auto-Update page for more information on this feature.
  • "Time Matching" allows you to apply one of a number of time matching schemes. See the Time Matching page for more detail on that.
  • "Set Time" allows you to adjust the current time in GARP. This is particularly useful when running in case study mode where the time in the application is not real time. You can reset the time either from one of the predefined time increments or by adjusting the clock.

Options are too numerous to list on this page and deserve their own page.

Hypertext help is available from this button.
