GARP Supported Data Types

GARP can read the following data formats for a given data type:

  • Satellite Imagery: MCIDAS Area Files, NOAAPort GINI format
  • Radar Data: NIDS (Level III), NOWrad® format
  • Gridded Data: GEMPAK format
  • Surface Data: GEMPAK format
  • Sounding Data: GEMPAK format
  • Wind Profiler Data: GEMPAK format

At the COMET Program, this data is acquired through a Local Data Manager (LDM) over the internet. Decoders spawned by the LDM convert the transmitted data to a file format utilized by GARP. These files are then stored on a local file server.
