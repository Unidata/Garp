GARP Supported Platforms

GARP is developed and tested on the following platforms:
  • Hewlett Packard 715, 755 running HPUX 10.x
  • Sun Sparc 20 running Solaris 5.5.1

When GARP is released, Unidata ports and tests it under the following operating systems:

  • Silicon Graphics Irix 5.3
  • IBM AIX 4.1
  • DEC OSF/1 3.0
  • DEC Ultrix 4.4
  • Linux Red Hat
