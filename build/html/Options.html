    Show Icon Bar

    This toggle button allows you to redraw the interface without an icon bar. This is particularly useful when you want to make the display window small but the icons in this icon bar prevent you from shrinking the interface. You can prevent the icon bar from displaying at startup using the -noicons command line argument.

    Show Status Bar

    This toggle button allows you to redraw the interface without a status bar at the bottom. You can prevent the status bar from displaying at startup using the -nostatus command line argument.
