GARP Geographic Areas

The buttons available under the Area button of the main menubar define a preset list of geographic areas from which you can choose. They are configurable in "Garp_defaults". Three specialized buttons of note are

  • DEFAULT which will reset the projection and area to the default projection and area for the current data type. These default projections are configurable in "Garp_defaults" on a per data type basis
  • Define which allows you to explicitly define a projection and area
  • Previous which allows you to redisplay the previous projection and area
Note that you can also define a geographic area with either of two zoom features using the left mouse button in the display window.
