GARP Overview

GARP provides a GUI around many of the capabilities of GEMPAK. Here is an overview of some of GARP's features.

General data display features include

  • Image data
    • Satellite (MCIDAS AREA and GINI formats)
    • Radar (NIDS/Level III and WSI NOWrad formats)
    • Images are displayed in the projection specified in the file (no remapping)
  • Model data
    • Can display any model data in GEMPAK file format
    • Display plan view, cross-section, time height and vertical profiles
    • Time match against analysis data or a forecast run
    • Run macros to create complex displays
  • Observed data (in GEMPAK file format)
    • METARs and any other observation data in a GEMPAK surface data file
    • Upper-air data
    • Wind profiler data
Any graphics data (model or observed) can be overlayed on any other data type in the same display mode (i.e., plan view on plan view, cross section on cross section)

User control of model diagnostic for display of vector and scalar data

  • User selection of level, contour or color fill interval, line types, and other general display characteristics via GUI control and user defined FDFs (Field Description Files)
  • Ability to type in a GEMPAK style GFUNC or GVECT
  • User selection for vector data symbols (barbs, arrows, streamlines)
  • Display contents of model data files in a text window
User control of layout of the station symbols
  • For upper air station symbol
  • For surface station symbol
Continuous data readouts from the cursor
  • Latitude and longitude
  • Raw pixel value for satellite and radar data
  • Reflectivity or velocity values for radar data
  • Brightness temperature for satellite data
  • Distance away from a selected point
GUI control over color enhancement tables for imagery
  • Selection of predefined color enhancement files
  • Dynamically roll the color bar
  • Invert color bar
  • Dynamically fade or increase the brightness of an image
Viewing Controls
  • Animate data with controls for loop speed and dwell
  • Control over geographic regions being displayed
    • User selection of predefined areas (North America, U.S., regions, Alaska, Hawaii, Canada)
    • Use keyboard to type in GEMPAK-style GAREA
    • Rubber-band box or centered zoom (redrawn using more data)
    • Frame editor lets user remove data frames from an animation loop
Overlay static plots (NEXRAD site names, radar range rings)

Can view either real-time or COMET case study data

Operating System Support

  • Tested on Solaris 2.x and HPUX 10.x
  • Ported to other supported platforms

    Caveats:

    • Currently, GARP has automatic time matching across data types. For example, if you choose to plot a 5-hour series of satellite images and then choose to overlay surface observations on top of the satellite images, the obs date/times which correspond to the selected satellite images will be pre-selected for you. You are free to change the date/times selected if you turn off time matching. See the time matching page for details.

    • Please note that currently, GARP will NOT allow you to combine multiple images. (For example, a satellite image and a radar image or an IR and a VIS satellite image.)
