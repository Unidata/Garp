GARP Installation Instructions

Installing GARP is pretty well documented in the installation notes below. You will need a computer running some brand of Unix on a supported platform to run GARP. You will also need some means of acquiring the data you want to view in one of the supported data types. Unless you are installing a standalone version of GARP complete with configuration tables, you will also need to install N-AWIPS first. That given, follow the instructions below.

This is the sixth general release of GARP, version 2.1.
