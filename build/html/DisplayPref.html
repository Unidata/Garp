GARP Display Preferences

Display Window Preferences


This dialog allows you to set several preferences for the entire display window.


Plot fills window area

This option allows you to control the extent of the area used in the drawing window. By default, GARP will try to use as much space in the window as it can for drawing. However, if this option is unset, GARP will use the exact specified geographic area. For example, if the user zooms (using the zoom box) over some area, the resulting area will be exactly the area in the zoom box.


Reset when loading images

By default, GARP performs a "reset" whenever the user loads an image. This means all the georeferencing information is released and the full image is displayed. The user can override this behavior by unsetting this option. No reset will occur and the image being loaded will display in the current geographic area (even if no part of the image covers the current area!).


Display partial frames

When plotting data, GARP normally updates the window after all the component data objects are finished drawing. That way, the user sees the entire frame plot at once. But, with this option set, data objects are displayed as they are plotted.
