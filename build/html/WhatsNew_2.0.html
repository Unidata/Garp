What's New in Version 2.0


 

GARP Version 2.0 is a major release with much added functionality. Some of the highlights are listed below.
 

 Major Functional Improvements

  • Time matching
  • Auto-Update
  • Grid macros
  • Graphic toggling
  • Map and Lat/Lon Grid backgrounds
  • On-Line Help
  • Full Image retained in RAM

Other Additions

  • Refresh button to manually update data lists
  • Automatic image color reloading (useful when multiple GARPS are sharing image colors)
  • Automatic pre-positioning of data time lists
  • Improved titles
  • Model-specific FDF capability
  • New command line options
  • Improved error diagnostics and user feedback
  • Many big fixes and memory leaks plugged
