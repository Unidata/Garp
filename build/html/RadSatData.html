Radar and Satellite Data

The appearance of this dialog will vary greatly with your data set. Generally, however, you'll see radio buttons inside scrolled windows that indicate the satellite or data source, the data scale, and finally the particular product type you are interested in. Below that is a scrolled window with date time stamps that match the criteria defined in the above scrolled windows. To the right of this, you'll see a toggle button that indicates whether you want a color bar displayed with the images or not. As with other data type dialogs, the Refresh button at the bottom of this dialog allows you to manually update the list of available times.

Starting with version 2.0 of GARP, image data is saved internally to provide future image combination capabilities.
