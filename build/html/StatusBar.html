GARP Status Bar Preferences

With this dialog you can configure what information is written to the status bar at the bottom of the GARP main window when data is displayed. You can display the current time, window coordinates, display coordinates, pixel value, and distance from a point. Toggle buttons allow you to choose which of these to display.

Also, you can adjust units for temperature and distance used in building the status bar label with option menus at the bottom of this dialog.

NOTE: Toggling off "Pixel Value" frees memory taken up by images loaded in GARP. If you are running GARP on a computer with less than 32 MB of memory or are otherwise saturating system resources (such as swap space), having this feature disabled may improve performance.
